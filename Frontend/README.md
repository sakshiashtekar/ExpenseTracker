ExpenseTracker is a simple mobile application (frontend + backend) built to let users track their incomes and expenses. The goal is to enable users to add, view, and manage their financial transactions in a clean, organized way.
1. Install dependencies - npm install
2. Start the app - cd Frontend -> npx expo start
   cd Backend -> npm run dev
4. Tech Stack -
   Frontend - React Native
   Authentication: Signup & login with Clerk
   Backend with Express: RESTful API connected to Neon-hosted Postgres
   Cloud Deployment: Hosted backend online for mobile access
   Rate Limiting: Protect API using Redis-based limiter
   API deployed on - Render.com
   Database - Neon
